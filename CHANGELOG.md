# Changelog

##[Update 2019/11/19]

1. Multi-GPU training added. The number of GPUs can be specified in the config file. See the example for more details.
2. Add a standard VoxCeleb example which uses the official training list (i.e. the VoxCeleb2 dev set).
3. Add a simple ResNet example which achieves EER 1.8% in the VoxCeleb test set.
